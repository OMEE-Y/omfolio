* Clean and responsive UI

##  Tech Stack

*  Next.js
*  Tailwindcss
*  Typescript



##  Installation & Setup

```bash
git clone https://github.com/OMEE-Y/omfolio.git
cd omfolio
npm install
npm run dev
```
